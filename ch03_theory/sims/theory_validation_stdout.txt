  Theoretical predictions:
    VI iterations ~ c * (1-gamma)^-1 for some constant c
    PI iterations ~ constant (typically 5-15)

     gamma    1/(1-g)   VI iters   PI iters      ratio
  --------------------------------------------------
     0.900       10.0         50          4       12.5

     0.995      200.0         59          6        9.8
     0.999     1000.0         60          5       12.0

  VI linear fit: iterations = 0.0053 * (1/(1-gamma)) + 55.5
  R^2 = 0.3025
  PI iterations range: 4-6

  Note: On this small gridworld (100 states), VI converges quickly for all gamma,
  so the linear scaling is weak. The O((1-gamma)^-1) bound is tighter in larger
  state spaces where the contraction rate dominates.
