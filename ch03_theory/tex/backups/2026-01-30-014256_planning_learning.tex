In this section, we connect Dynamic Programming (DP) with Reinforcement Learning (RL) methods through a concrete navigation example. Both methods attempt to find optimal policies in Markov Decision Processes (MDPs). An MDP is defined as a tuple $(S, A, P, R, \gamma)$, where $S$ denotes the state space, $A$ the action or choice space, $P(s'|s,a)$ the transition probability function, $R(s,a,s')$ the reward or utility function, and $\gamma \in [0,1]$ the discount factor. Policies $\pi$ are conditional probabilities functions that tell us the probability of taking actions in states. 

Following Sutton and Barto's formulation, Generalized Policy Iteration (GPI) refers to the interaction between policy evaluation and policy improvement processes. Policy evaluation makes the value of being in states, i.e. the an estimate of the expected discounted returns, when following the current policy. Policy improvement finds the best policy given a valuation of the states. When both processes stabilize, the resulting policy and value function is optimal.

\subsubsection{Dynamic Programming}

First, we look at two DP techniques. The solution is stored in value and policy tables, initialized at 0. which sweep through, updating all states at each iteration (full-width). They both look ahead one step (depth=1), and use the previous guess of the value function to update (bootstrapping). The learning rate is 1, i.e. it is a full update. At each iteration, Value Iteration (VI) computes the optimal value function through the Bellman optimality equation:
\begin{equation}
\text{Value Iteration:\quad} \forall s, V(s) \leftarrow \max_a \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V(s')]
\end{equation}

The max operator delivers policy improvement, while the assignment delivers value evaluation. By requiring transition probabilities and potential rewards across all actions and future states, VI is inherently model-based. Theoretically, VI converges to the optimal value function at a linear rate, with error decaying geometrically with ratio $\gamma$ \cite{bertsekas1996}. For any finite MDP with $\gamma < 1$, VI guarantees convergence to $V^*$ from any initial $V_0$ \cite{sutton2018}.

Policy Iteration (PI) Alternates between policy evaluation and improvement:
\begin{align}
 \text{Policy Iteration:\quad} V^{\pi}(s) &= \sum_{s'} P(s'|s,\pi(s))[R(s,\pi(s),s') + \gamma V^{\pi}(s')] \\
\pi'(s) &= \argmax_a \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V^{\pi}(s')]
\end{align}

This method separates policy evaluation and policy improvement, and begins with a random policy, and recursively finds the value of states for it. PI converges to an optimal policy in a finite number of iterations, with each policy improvement yielding a strictly better policy until optimality is reached \cite{bertsekas1996}.


\subsubsection{Reinforcement Learning}

We now turn to Reinforcement Learning (RL) methods, which do not require knowledge of $P$ and $R$ functions but only require access to transitions and rewards only at those particular actions and states (i.e. a simulator). The first two are temporal-difference methods that learn value tables. They begin from an initial state and explore their environment one state at a time (width=1) and look ahead one step (depth=1), using their previous estimate of the value function (bootstrapping). The actions are not selected greedily, but $\epsilon$-greedily which allows for randomness and exploration of the state space. The updates also happen incrementally with the learning rate $\alpha<1$.  

Q-learning is an off-policy algorithm that updates action-values using individual sampled experiences, following the update rule:
\begin{equation}
\text{Q-learning:\quad} Q(s,a) \leftarrow Q(s,a) + \alpha [R + \gamma \max_{a'} Q(s',a') - Q(s,a)]
\end{equation}

\paragraph{Q-Learning as Stochastic Value Iteration}
We can formally demonstrate that Q-learning is a stochastic approximation of value iteration. Let us define the optimal Q-factor, $Q^*(i,u)$, as the expected discounted cost of taking action $u$ in state $i$ and following the optimal policy thereafter. The Bellman optimality equation for Q-factors is:
\begin{equation}
Q^*(i,u) = \sum_j p_{ij}(u) \left( g(i,u,j) + \alpha \min_{v} Q^*(j,v) \right)
\label{eq:q-bellman}
\end{equation}
This equation suggests a value iteration algorithm for Q-factors:
\begin{equation}
Q_{k+1}(i,u) = \sum_j p_{ij}(u) \left( g(i,u,j) + \alpha \min_{v} Q_k(j,v) \right)
\label{eq:q-vi}
\end{equation}
In a model-free setting, the transition probabilities $p_{ij}(u)$ are unknown, so the expectation (the summation over $j$) cannot be computed. Q-learning approximates this expectation using a single sample $(s', r)$ obtained from taking action $u$ in state $i$. The Q-learning update rule can be written as:
\begin{equation}
Q_{k+1}(i,u) = (1-\gamma_k)Q_k(i,u) + \gamma_k \left( r + \alpha \min_{v} Q_k(s',v) \right)
\label{eq:q-learning-update}
\end{equation}
where $\gamma_k$ is the learning rate. This is a stochastic approximation of the value iteration update in Eq. \ref{eq:q-vi}. As shown in \citet{bertsekas1996}, this can be analyzed as a stochastic iterative algorithm. The update can be expressed as $Q_{k+1} = (1-\gamma_k)Q_k + \gamma_k(HQ_k + w_k)$, where $H$ is the Bellman operator and $w_k$ is a zero-mean noise term. The convergence of Q-learning is guaranteed if the operator $H$ is a contraction mapping, which is the case for discounted problems.

At each step, the agent observes a tuple $(s,a,s',r)$, chooses actions using an $\varepsilon$-greedy policy (selecting random actions with probability $\varepsilon$ and the best-known actions with probability $1-\varepsilon$), and updates its value estimates. Over time, $\varepsilon$ typically decreases, shifting from exploration to exploitation. Under conditions of infinite exploration, diminishing learning rates, and greedy-in-the-limit behavior, Q-learning converges to the optimal action-value function $Q^*$ with probability 1 \citep{watkins1992_paper}.

SARSA for State-Action-Reward-State-Action updates action-values using the action selected by the current policy for the next state:
\begin{equation}
\text{SARSA:\quad} Q(s,a) \leftarrow Q(s,a) + \alpha [R + \gamma Q(s',a') - Q(s,a)]
\end{equation}

Unlike Q-learning, SARSA computes the Bellman error based on actual behavior (next action chosen by the policy) and not greedy-behavior, making it on-policy i.e. the same policy that generated the samples is being updated. It similarly uses $\varepsilon$-greedy exploration to generate samples $(s,a,s',r,a')$. SARSA converges to the optimal $Q^*$ with probability 1 when coupled with a policy that becomes greedy in the limit, provided every state-action pair is visited infinitely often \cite{singh2000}.


Monte Carlo methods are on-policy methods that learn from complete episodes, updating action-values based on observed returns:
\begin{equation}
\text{Monte Carlo:\quad} Q(s_t,a_t) \leftarrow Q(s_t,a_t) + \alpha[G_t - Q(s_t,a_t)]
\end{equation}

They begin from random policies, which are traced out till termination (reaching the goal or being truncated), and then updates are made to each state-action visited. Here, the return $G_t = \sum_{k=0}^{T-t-1} \gamma^k R_{t+k+1}$ represents cumulative discounted rewards from time $t$ to the episode's end. This does use bootstrapping: it does not estimate value using existing estimates like $V(s')$ or $Q(s',a')$. Instead, it waits for the full return to be observed before making updates, making the updates unbiased. For episodic tasks with exploring starts, Monte Carlo methods converge to the optimal policy as the number of episodes approaches infinity \cite{tsitsiklis2002}.

Lastly, we touch on policy-gradient methods. REINFORCE directly optimizes the policy parameters $\theta$ based on observed episode returns $G$, following the gradient direction that increases the probability of actions leading to higher rewards. The policy parameterized by, typically, a neural network:

\begin{equation}
\text{REINFORCE:\quad} \theta \leftarrow \theta + \alpha \nabla_\theta \log \pi_\theta(a|s) G
\end{equation}

One can interpret the REINFORCE as trying to generate data from a policy and re-estimating the policy parameters using maximum likelihood via gradient ascent, but with increased weight when the trajectory returns larger returns. This step pulls the policy towards generating larger returns. REINFORCE converges to a stationary point of the expected return function with probability 1 \citep{williams1992_reinforce}. In tabular settings with suitable exploration, it achieves global optimality at a geometric rate \cite{agarwal2020}.

\subsection{Deep Reinforcement Learning}

Deep Q-Network (DQN) extends Q-learning by approximating the action-value function with a deep neural network \(Q(s,a;\theta)\). Instead of storing values in a table, DQN uses a parameterized model trained on transition samples \((s,a,r,s')\) drawn from a replay buffer. Its objective is to minimize the temporal-difference (TD) error:

\[
L(\theta) \;=\; \mathbb{E}_{(s,a,r,s')}\!\Bigl[\bigl(r \,+ \,\gamma \,\max_{a'} Q(s',a';\theta^-)\;-\;Q(s,a;\theta)\bigr)^2\Bigr].
\]

Here, \(\theta^-\) denotes the parameters of a separate target network, periodically synchronized to prevent instability from moving targets. By replaying past transitions and using a target network, DQN decouples correlated updates and stabilizes learning, enabling it to scale to high-dimensional state spaces. Under proper conditions including Lipschitz-continuous Q-function and sufficient exploration, DQN converges to an approximate Bellman fixed point \cite{Ramaswamy2021}. With experience replay and target networks breaking the "deadly triad" instability, DQN can achieve geometric convergence to $Q^*$ when the network class can represent it \cite{Zhang2023}.

\subsection{Example 2: Capital Replacement}
We study a maintenance optimization problem with $N$ parallel engines formulated as a Markov Decision Process. Each engine $i$ has mileage state $m_i \in \{0,1,2,3,4,5\}$, with system state $s = (m_1,...,m_N)$. The action space $A(s)$ consists of all subsets of engines to replace, constrained by capacity $C=3$. The cost function is:
\begin{equation}
c(s,a) = \alpha \sum_{i=1}^{N} \mathbf{1}\{m_i > 0\} + \beta |a|
\end{equation}
where $\alpha=1.0$ is the cost of aged engines, $\beta=5.0$ is the replacement cost, and $|a|$ is the number of replaced engines. State transition is deterministic:
\begin{equation}
m_i' =
\begin{cases}
0 & \text{if } i \in a \\
\min(5, m_i+1) & \text{otherwise}
\end{cases}
\end{equation}
The objective is to find policy $\pi^*$ minimizing expected discounted cost with $\gamma=0.95$:
\begin{equation}
V^*(s) = \min_{\pi} \mathbb{E}_{\pi} \left[ \sum_{t=0}^{\infty} \gamma^t c(s_t, \pi(s_t)) \right]
\end{equation}

For the DQN implementation, we map the system state $s = (m_1,\ldots,m_N)$ directly to the neural network input\footnote{The neural network architecture consists of fully-connected layers (128-64) with ReLU activations, trained using Adam optimizer with learning rate 1e-3 and batch size 64. To ensure stability, we employ $\epsilon$-greedy exploration starting at $\epsilon=1.0$ and decaying by a factor of 0.995 per episode to a minimum of 0.01, along with a target network updated every 50 episodes.}, where each mileage value $m_i$ becomes a feature in the input vector. The action space is encoded as indices of a fixed-size output vector, where index $j$ corresponds to the $j$-th possible subset of engines to replace (e.g., empty set, single engines, pairs, and triples up to capacity $C=3$). 

Dynamic Programming (DP) solves this exactly via value iteration, but faces exponential state growth $\mathcal{O}(6^N)$. For each state, DP evaluates all possible actions and selects the one minimizing the Bellman equation until convergence, measured by maximum value change between iterations falling below 1e-5. DQN addresses the scalability challenge by approximating the value function and learning from sampled experiences, with convergence measured through the Bellman residual—the average absolute difference between the current Q-value estimate and the target.

\subsection{Performance of Reinforcement Learning on Optimization Problems}

Multiple empirical studies provide evidence that Deep RL can indeed overcome the curse of dimensionality that limits traditional Dynamic Programming. \cite{nomura2025} showed that for perishable inventory control with dynamic pricing, PPO achieved near-optimal solutions with a 75\% reduction in computation time compared to DP, with authors explicitly noting that increasing problem dimensionality would render DP impractical while DRL would remain viable. \cite{sabri2024} demonstrated that attention-based DRL delivered maintenance schedules with 30.5\% lower costs than DP-based heuristics on combinatorial problems, computing in seconds what DP required over an hour to solve. For high-dimensional inventory problems where exact DP was infeasible, \cite{boute2022} showed that A3C matched the performance of the best specialized heuristics and approximate DP methods across three inventory settings. These preliminary results show that while Deep RL requires substantial initial training experiences and hyperparameter tuning, they can effectively approximate optimal policies in high-dimensional state spaces where exact DP becomes computationally prohibitive.

\subsection{Theoretical Foundations Connecting DP and RL (2020--2025)}

The relationship between Dynamic Programming and Reinforcement Learning has become mathematically precise in recent years, with tight sample complexity bounds, convergence guarantees, and formal equivalence results now available across multiple settings relevant to economics. We survey the key theoretical advances organized by theme.

\subsubsection{Convergence and Sample Complexity: Q-Learning Meets Minimax Optimality}

The most significant recent theoretical advance establishes \textbf{tight sample complexity bounds} that precisely characterize when RL algorithms match DP's efficiency. \citet{li2024minimax} prove that vanilla Q-learning is strictly suboptimal, requiring $\tilde{O}(SA/(1-\gamma)^5\varepsilon^2)$ samples---worse than the minimax lower bound $\Omega(SA/(1-\gamma)^3\varepsilon^2)$ by a factor of $(1-\gamma)^{-1}$. This gap arises from overestimation bias in bootstrapping, connecting directly to the Bellman optimality equation's maximum operator.

\citet{bhandari2021finite} establish finite-time convergence rates for TD learning with linear function approximation by connecting TD updates to online gradient descent. Their analysis extends to TD($\lambda$) and Q-learning for optimal stopping, providing explicit non-asymptotic bounds under both i.i.d.\ and Markovian sampling. For average-reward settings, \citet{lee2025optimal} characterize optimal $O(1/k)$ convergence rates for Value Iteration, establishing span-based complexity that matches lower bounds within a factor of 8.

\citet{carvalho2020convergence} address the ``deadly triad'' directly, proving convergence of modified Q-learning with linear function approximation under general assumptions. Their work establishes rigorous bounds connecting the Q-learning fixed point to the DP optimal Q-function---a result particularly relevant for econometric applications using linear approximations.

Additional contributions include \citet{he2021nearly}, who establish nearly minimax optimal RL for discounted MDPs, and \citet{rowland2024quantile}, who provide a rigorous analysis of quantile temporal-difference learning.

\subsubsection{Model-Based Methods Provably Dominate in Stationary Environments}

A striking theoretical consensus has emerged: \textbf{model-based RL (closer to DP) achieves optimal sample complexity} while model-free methods typically incur polynomial overhead. \citet{zhang2024settling} settle a long-standing open problem by proving that model-based algorithms achieve minimax-optimal regret $\tilde{O}(\sqrt{SAH^3K})$ without any burn-in cost---eliminating the warmup requirements that plagued model-free methods.

\citet{li2024offline} prove that the model-based plug-in approach achieves minimax-optimal sample complexity for offline RL with distribution shift, achieving $SC^*/(1-\gamma)^3\varepsilon^2$ where $C^*$ is a concentrability coefficient. Their companion paper \citep{li2024breaking} breaks the $S^2A$ sample size barrier, showing model-based algorithms need only $\tilde{O}(SA/(1-\gamma)^3\varepsilon^2)$ samples.

\citet{zhu2024representation} provide a novel \textbf{circuit complexity explanation} for these gaps: transition and reward functions can be represented by constant-depth polynomial-size circuits, while optimal Q-functions require exponential circuit complexity. This fundamental representation asymmetry explains model-based advantages from first principles.

The one exception is non-stationary environments. \citet{mao2021nonstationary} prove that model-free methods achieve superior dynamic regret $\tilde{O}(S^{1/3}A^{1/3}\Delta^{1/3}HT^{2/3})$ when dynamics change, since model estimation becomes a liability---an important consideration for econometric applications with structural breaks.

\subsubsection{Value Equivalence: When Learned Models Suffice for Planning}

The theoretical foundations for combining DP-style planning with learned models have matured considerably. \citet{grimm2020value} introduce the \textbf{value equivalence principle}: two models are value equivalent if they yield identical Bellman operator updates for a given set of policies and value functions. As these sets expand, the class of value-equivalent models contracts to the true model---but crucially, smaller sets permit useful approximations.

\citet{grimm2021proper} extend this with \textbf{proper value equivalence (PVE)}, proving that PVE models are sufficient for planning even when they differ from ground truth. They construct a loss function for learning PVE models and demonstrate that MuZero minimizes an upper bound of this loss, providing the first rigorous justification for MuZero's empirical success.

\citet{ayoub2020value} introduce value-targeted regression, proving regret bounds that depend on value function structure rather than state space size. Learning models to minimize value prediction error (rather than maximum likelihood) yields better sample complexity---a decision-aware approach directly relevant to structural estimation where the goal is policy evaluation, not dynamics prediction.

\citet{lobel2024simulation} provide optimally tight bounds for the \textbf{simulation lemma}, the foundational result quantifying value-prediction error under model misspecification. Their bound is sub-linear in transition function misspecification, improving error propagation analysis for all model-based methods. Additional contributions include \citet{janner2019model} on deciding when to trust learned models, and \citet{grill2020planning} on gap-dependent sample complexity for planning.

\subsubsection{Neural Network Approximation: Bridging Approximate DP and Deep RL}

The theoretical gap between classical approximate DP and deep RL has narrowed substantially. \citet{fan2020dqn} provide the first rigorous analysis of DQN, establishing that neural Fitted Q-Iteration converges geometrically to the optimal Q-function up to intrinsic statistical error induced by \textbf{ReLU network approximation bias} and finite sample size. They provide theoretical justification for experience replay and target networks.

\citet{zhang2021target} prove that target networks can break the deadly triad by proposing Polyak-averaging updates with projections that converge to regularized TD fixed points. \citet{gaur2023fqi} establish \textbf{order-optimal sample complexity} $\tilde{O}(1/\varepsilon^2)$ for Fitted Q-Iteration with two-layer ReLU networks without requiring linear or low-rank MDP structure.

\citet{wang2020eluder} establish the first provably efficient RL algorithm with general nonlinear value function approximation, achieving regret $\tilde{O}(\mathrm{poly}(dH)\sqrt{T})$ where $d$ depends on \textbf{eluder dimension} and log-covering numbers---a complexity measure that extends beyond linear settings while remaining tractable.

On estimation bias, \citet{ren2021underestimation} provide rigorous analysis of underestimation bias in Double Q-learning, while \citet{lan2020maxmin} show how Maxmin Q-learning can systematically control bias from positive to negative. These results connect to DP's Bellman operator properties. \citet{lim2024regularized} contribute further with analysis of regularized Q-learning.

\subsubsection{Generalized Policy Iteration Achieves Linear Convergence}

The interaction between policy evaluation and policy improvement---the core of both DP and RL---is now understood through the lens of \textbf{gradient domination properties}. \citet{agarwal2021theory} establish global convergence for policy gradient using a Polyak-like gradient domination property, introducing the distribution mismatch coefficient framework that governs GPI convergence rates.

\citet{xiao2022convergence} develops weak gradient-mapping dominance theory, proving $O(1/k)$ convergence for projected policy gradient and showing policy mirror descent achieves linear convergence with geometrically increasing step sizes---without entropy regularization. Crucially, as step sizes approach infinity, policy mirror descent reduces to classical Policy Iteration, establishing a continuum between gradient-based and value-based methods.

\citet{cen2022fast} prove non-asymptotic linear convergence (plus local quadratic convergence) for entropy-regularized Natural Policy Gradient, with iteration complexity $O((1/(1-\gamma))\log(1/\varepsilon))$. \citet{wu2020twotimescale} provide the first non-asymptotic analysis of \textbf{two-timescale actor-critic} with single Markovian sample paths, establishing $O(\varepsilon^{-2.5})$ sample complexity.

For deep networks, \citet{li2023actorcritic} extend actor-critic convergence from shallow to arbitrary-depth networks, showing convergence to neighborhoods where average squared gradients scale as $\tilde{O}(1/\sqrt{m}) + O(\varepsilon)$. \citet{muller2024npg} analyze NPG as gradient flows in Hessian geometries, proving local quadratic convergence when step sizes equal inverse regularization strength. Additional foundational work includes \citet{geist2019regularized} on regularized MDPs.

\subsubsection{Synthesis: Implications for Economics}

The theoretical literature reveals several key insights for integrating RL into economic methods.

\paragraph{Model-based approaches align with economic estimation.} The proven optimality of model-based RL \citep{zhang2024settling,li2024breaking} supports econometric approaches that estimate structural primitives before solving for optimal policies. The $(1-\gamma)^{-1}$ factor advantage over model-free methods becomes substantial for patient agents with high discount factors---precisely the long-horizon settings common in lifecycle models.

\paragraph{Value equivalence relaxes full-model requirements.} The value equivalence principle \citep{grimm2020value} suggests that structural models need not perfectly replicate all dynamics---only those relevant for policy evaluation. This provides theoretical grounding for parsimonious structural specifications that capture decision-relevant features without full realism.

\paragraph{Function approximation bounds inform practical choices.} The eluder dimension framework \citep{wang2020eluder} and neural network convergence results \citep{fan2020dqn,gaur2023fqi} provide guidance on approximation architectures. Linear function approximation retains strong guarantees \citep{bhandari2021finite}, while neural approximations require careful bias control.

\paragraph{GPI theory unifies estimation and optimization.} The gradient domination framework establishes that policy evaluation and improvement can be interleaved without losing convergence guarantees---supporting iterative estimation-optimization procedures common in structural work. The continuum between policy gradient and policy iteration \citep{xiao2022convergence} offers flexibility in algorithm design.

\paragraph{Distribution mismatch matters for counterfactuals.} The distribution mismatch coefficient \citep{agarwal2021theory} and concentrability coefficients \citep{li2024offline} quantify how off-policy evaluation degrades with distribution shift---directly relevant for counterfactual policy analysis where target policies differ substantially from observed behavior.
